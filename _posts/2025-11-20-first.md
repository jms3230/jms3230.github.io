# 첫 포스팅
## Jekyll로 깃허브 페이지 만들기
1. https://github.com/topics/jekyll-theme 에서 테마 선택
2. 선택한 테마의 Repository를 내 계정으로 Fork
3. Fork한 Repository의 이름을 내깃허브아이디.github.io로 설정
4. config.yml에서 url 부분 변경
  - url : "내깃허브아이디.github.io"
5. 포스팅
  1. Repository 우상단 Add File -> Create new file
  2. _posts/20xx-mm-dd-원하는단어.md
  3. 내용은 마크다운으로 작성
  4. Commit changes...
6. 리포지토리 세팅
   1. 상단 Settings->좌측 Pages->Build and deployment를 Github Actions로 변경
7. --- layout: home # Index page --- 나타나는 문제 해결중...

## Docker Container에 Clone 후
- VS Code 활용
- tools의 init.sh를 실행하고 나니 초기 파일들 다 사라짐
- 다시 config와 post 작성
- push가 안됨 - 과거 버전이 됨
- 다시 pull 하고 적당히 merge 후 push
- Action에 멈춰있는 것들이 있어서 수동으로 실행했더니 페이지 등장
